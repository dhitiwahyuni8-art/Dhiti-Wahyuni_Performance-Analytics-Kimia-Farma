# Dhiti-Wahyuni_Performance-Analytics-Kimia-Farma
SQL query for Kimia Farma performance analytics (2020-2023) | BigQuery | Rakamin Academy Final Project

SQL query untuk membuat tabel analisa_transaksi dari dataset Kimia Farma periode 2020–2023.

## Query
- `analisa_transaksi` — membuat tabel analisis dengan 
  menggabungkan 4 tabel sumber menggunakan LEFT JOIN dan 
  kalkulasi nett sales, nett profit, serta persentase gross laba.
